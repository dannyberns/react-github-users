## Installation

Install github-users with npm

```bash
  # Clone this repository
  $ git clone https://github.com/dannyberns/react-github-user.git

  # Go into the repository
  $ cd react-github-user

  # Install dependencies
  $ npm install

  # Run the app
  $ npm start
```
